# 🚰 Wasserschacht – AutoCAD Projekt

## 📖 Beschreibung

Dieses Projekt umfasst die vollständige Konstruktion eines Wasserschachts, erstellt mit AutoCAD. Es beinhaltet technische Zeichnungen, Schnitt- und Detailansichten sowie eine Stückliste aller verbauten Komponenten.

## 🔧 Inhalte

- Grundriss
- Schnittzeichnung
- Detailzeichnungen
- Rohrleitungsführung
- Schachtabdeckung
- Steigleiter mit Einsteighilfe
- Wandhalterung
- Stückliste
- Bemaßung nach technischen Standards

## 🛠️ Verwendete Software

- AutoCAD

## 📂 Projektdateien

- PDF-Zeichnung


## 🎯 Projektziel

Planung und Konstruktion eines funktionalen Wasserschachts für die kommunale Wasserversorgung unter Berücksichtigung von Wartung, Sicherheit und technischen Anforderungen.
